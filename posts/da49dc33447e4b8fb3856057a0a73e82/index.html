1.目标
人物档案：
工资：5000 元/月
开销：2000 元/月
保险： 500 元/月
目标：资产增值与保值
2. 每月投资基数
存量资金
( 存款 - 应急金 ) / 20 = 0
每月增量资金
工资 - 开销 - 保险 = 2500
风险系数
100 - 25 = 75%
每月投资基数
( 存量 + 增量 ) * 75% = 1875
3. 要投资的基金
昨日温度，最低的宽基指数
上证50 ~ 17.5°
中证500 ~ 17.5°
随意选择一只跟踪上证50 指数的基金
4. 确定买入和卖出
买入的基数为每月投资基数。
卖出的基数为当前持有总量。

5. 切换指数
出现另一只指数的温度低于当前指数2°时，切换到后者。
前者的温度只要不超过 40°，就持续持有。
若所有指数温度都超过 40°，资金用于别的投资，或存入货基等待机会。
