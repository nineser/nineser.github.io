脱离文档流
如果父级设置 display:none; 子级设置 display:block 不会显示
元素在隐藏前并不都是 block，需要提前存储元素的 display 值
visibility
visibility: hidden
不脱离文档流
如果父级设置 visibility:hidden; 子级设置 visibility:visible 时子级会显示
hidden
HTML 的全局属性 hidden
脱离文档流
btm.onclick = function(){ element.removeAttribute('hidden'); }
JavaScript
overflow
通过利用父级的 overflow:hidden 配合父级的 height:0 或 width:0 来实现元素的显隐
[注意] 设置 overflow 的元素在绝对定位元素和其包含块之间时，overflow 会失效
<style> #testWrap{ height: 70px; transition: height 1s; overflow: hidden; } </style> <button id="show">显示</button> <button id="hide">隐藏</button> <div id="testWrap"> <div id="test" style="background:lightblue; width:100px;height:60px; margin-top: 10px;"> 测试内容 </div> </div> <script> show.onclick = function(){ testWrap.style.height = '70px'; } hide.onclick = function(){ testWrap.style.height = '0'; } </script>
HTML
transform
transform 是一些效果的集合，主要是移动，旋转，缩放和倾斜。
通过不同的变形函数可以实现元素的显隐效果。
[注意]IE9-浏览器不支持，safari3.1-8、android2.1-4.4.4、IOS3.2-8.4都需要添加前缀
<button id="show">显示</button> <button id="hide">隐藏</button> <div id="test" style="background:lightblue; width:100px;height:60px; margin-top: 10px; transition: 0.5s;"> 测试内容 </div>
HTML
transform:scale(0)
show.onclick = function(){ test.style.transform ='scale(1)'; } hide.onclick = function(){ test.style.transform ='scale(0)'; }
JavaScript
transform: rotateX
show.onclick = function(){ test.style.transform ='rotateX(0)'; } hide.onclick = function(){ test.style.transform ='rotateX(90deg)'; }
JavaScript
transform:skew(90deg)时，元素被隐藏
show.onclick = function(){ test.style.transform ='skew(0)'; } hide.onclick = function(){ test.style.transform ='skew(90deg)'; }
JavaScript
覆盖
利用定位元素可以覆盖普通流元素的特性。
为元素的before伪元素设置相同的尺寸，通过控制伪元素的定位属性，实现显隐效果。
<style> #test:hover:before{ content: ""; position: absolute; width: 100px; height: 60px; background-color: white; } </style> <div style="background: lightblue; width:100px;height:60px; margin-top: 10px;">测试内容</div>
HTML
偏移
另一种常见思路是偏移，将元素移动到视窗范围外，也可以实现等价的显隐效果
<button id="show">显示</button> <button id="hide">隐藏</button> <div id="test" style="background:lightblue; width:100px;height:60px; margin-top: 10px; transition: 0.5s;"> 测试内容 </div>
