默认样式
table{ border-collapse: separate; border-spacing: 2px; border: 1px solid gray; }
CSS
常用样式
border-spacing: x y /* x 为 水平间距 */ /* y 为 垂直间距 */ /* 若只有一个值，则水平间距和垂直间距相等。*/ /* 不可为负值 */
CSS
empty-cells: show(default) empty-cells: hide /* 不在空单元格周围绘制边框和背景，类似于hidden效果 */
CSS
💡
CSS实际上有两种截然不同的边框模型。 1. 如果单元格相互之间是分隔的，是分隔边框模型在起作用； 2. 另一种是合并边框模型，单元格边框会相互合并。
/*在分隔边框模型中，不能为行、行组、列和列组设置边框*/ border-collapse:separate; /*在合并边框模型中，表格无法设置内边距padding，且单元格边框之间也没有间距。*/ border-collapse:collapse;
CSS
table-layout: auto /*自动宽度*/ table-layout: fixed /*固定宽度*/
CSS
💡
对于表单元格的长文本来说， - 使用word-wrap或word-break来强制换行， - 使用text-overflow实现文本溢出控制 - 都需要设置 table-layout:fixed
vertical-align: top;//顶端对齐 vertical-align: bottom;//底端对齐 vertical-align: middle;//中间对齐 vertical-align: baseline(默认);//基线对齐
CSS
💡
vertical-align: sub-top-bottom 应用到表格单元格时会被忽略
组成元素
<tr>、<th>、<td>
colspan: 单元格横跨的列数
rowspan: 单元格横跨的行数
<col> 为表格中一个或多个列定义属性值
<colgroup> 对表格中的列进行组合，以便对其进行格式化
属性 span 规定 col 元素应该横跨的列数
样式
visibility: collapse：该列或列组的所有单元格不显示(设置为其他值则无效)
border：只有当border-collapse:collapse时，才能设置border
background：只有当单元格及其行有透明背景时，列或列组的背景才可见
width：定义列或列组的最小宽度
<table border="1" style="border-collapse: collapse"> <colgroup span="2" style="width:100px; background-color: red"></colgroup> <col style="background-color: green; width:200px; border: 3px solid blue;" > <tr> <td>数字</td> <td>中文</td> <td>英文</td> </tr> <tr> <td>1</td> <td>一</td> <td>a</td> </tr> <tr> <td>2</td> <td>二</td> <td>b</td> </tr> </table>
HTML
<thead>、<tbody>、<tfoot>
<table border="1" > 　 <caption style="caption-side:bottom">北京天气</caption> <thead> <tr> <th>地区</th> <th>天气</th> </tr> </thead> <tfoot> <tr> <td>北京</td> <td>都雾霾</td> </tr> </tfoot> <tbody> <tr> <td>城八区</td> <td>雾霾</td> </tr> <tr> <td>郊区</td> <td>雾霾</td> </tr> </tbody> </table>
HTML
表格层
CSS定义了 6 个不同的层，对应表各个方面的样式都在其各自的层上绘制。
默认地，所有元素背景都是透明的，
如果单元格、行、列等没有自己的背景，则table元素的背景将透明这些内部元素可见。
操作表格
HTML DOM 为 <table>、<tbody>、<tr>元素添加了属性和方法。
为 table 添加属性和方法
caption // caption 元素 tHead tBodies // tbody 元素的 HTMLCollection tFoot createCaption(); createTHead(); createTFoot(); deleteCaption(); deleteTHead(); deleteTFoot();
JavaScript
为 tbody 添加属性和方法
rows deleteRow(pos); insertRow(pos);
JavaScript
为 tr 添加属性和方法
cells deleteCell(pos); insertCell(pos);
JavaScript
创建表格
//创建表格 var table = document.createElement('table'); table.border = '1'; table.width = '100%'; // 创建 tbody var tbody = document.createElement('tbody'); table.appendChild(tbody); // 创建 rows[0] tbody.insertRow(0); tbody.rows[0].insertCell(0); tbody.rows[0].cells[0].appendChild(document.createTextNode('cell 1,1')); tbody.rows[0].insertCell(1); tbody.rows[0].cells[0].appendChild(document.createTextNode('cell 2,1')); //将表格插入文档主体 document.body.appendChild(table);
