匹配原理
浏览器先产生一个元素集合，这个集合通常由最后一个部分的索引产生
然后向上匹配，如果不符合上一个部分，就把该元素从集合中删除
直到整个选择器都匹配完，此时还在集合中的元素就是匹配这个选择器的集合
基本选择器
id、class、element、伪类
父代、子代、并列
后代选择器
A B
A>B ：即 B 是 A 的直接子节点
属性选择器
语法 []
属性和值（presence and value）
[attr] - 具有 attr 属性
[attr = val] - attr 属性被赋为 val 的元素
[attr ~= val] - 具有 attr 属性，val 为 attr 值包含的被空格分隔的取值列表中的一个
子串值 （Substring value）
[attr |= val] 值是 val 或值以 val- 开头
[attr ^= val] val 开头，包括 val
[attr $= val] val 结尾，包括 val
[attr *= val] 包含子字符串 val
结构伪类
:nth-child(n) 即 第 n 个 子元素
:nth-of-type(n) 即 第 n 个 特定类型的子元素
除了 E:first-child ，其它都需 IE9+ 以上
:root
选择文档的根元素，即元素
E:first-child
属于父元素的第一个子元素，且该子元素是 E 类型
与 E:nth-child(1) 等同
IE6- 不支持，且 IE8- 需 doctype
E:last-child
属于父元素的最后一个子元素，且该子元素是 E 类型，
与 E:nth-last-child(1) 等同
E F:nth-child(n)
选择父元素的第 n 个子元素，且父元素是 E，子元素是 F 类型
E F:nth-last-child(n)
选择父元素的倒数第 n 个子元素，且父元素是 E，子元素是 F 类型
E F:nth-of-type(n)
选择父元素的第 n 个 F 类型的子元素，且父元素是 E 类型
E F:nth-last-of-type(n)
选择父元素的倒数第 n 个 F 类型的子元素，且父元素是 E 类型
E F:first-of-type
选择父元素的第 1 个 F 类型的子元素，与 E:nth-of-type(1) 相同
E F:last-of-type
选择父元素的最后 1 个 F 类型的子元素，与 E:nth-last-of-type(1) 相同
E F:only-child
选择 E 仅有的一个子元素，且该子元素是 F 类型
E F:only-of-type
选择 E 的一个 F 类型子元素，且该元素是 E 唯一的 F 类型子元素
E F:empty
选择 E 的 F 类型子元素，且这些元素是空标签
伪元素
伪装成元素，但不是元素，这与生成内容相关。
生成内容主要指由浏览器创建的内容，而不是由标志或内容来表示。
生成内容主要由 :before 和:after伪元素来实现，还包括:first-line,:first-letter 和::selection
:first-letter
指定一个元素第一个字母的样式
\[注意 1\]所有前导标点符号应与第一个字母一同应用该样式
\[注意 2\]只能与块级元素关联
\[注意 3\]只有当选择器部分和左大括号之间有空格时，IE6-浏览器才支持。
div:first-letter { color: red; }
CSS
:first-line
设置元素中第一行文本的样式
\[注意 1\]只能与块级元素关联
\[注意 2\]只有当选择器部分和左大括号之间有空格时，IE6-浏览器才支持
div:first-line { color: red; }
CSS
:before \(IE7-浏览器不支持\)
在元素内容的最开始插入生成内容
\[注意\]默认这个伪元素是行内元素，且继承元素可继承的属性；IE7-浏览器中必须声明!DOCTYPE，否则不起作用
::section \(IE8-不支持\)
匹配被用户选择的部分
\[注意 1\]firefox 浏览器需要添加-moz-前缀
\[注意 2\]只支持双冒号写法
\[注意 3\]只支持颜色和背景颜色两个样式
div::selection { color: red; }
CSS
content 属性
content 属性应用于 before 和 after 伪元素
content: normal; (默认)content: <string>|<uri>|attr(<identifier>);
CSS
\[string\]
<string> 里面的内容会原样显示，即使包含某种标记也不例外。
\[注意 1\]如果希望生成内容中有一个换行，则需要使用\A
\[注意 2\]若是一个很长的串，需要它拆分成多行则需要用\对换行符转义
div:before { content: '第一段\\ 第二段'; } div:after { content: '\\A后缀'; }
CSS
\[uri\]
div:before { content: url('arrow.gif'); }
CSS
\[attr\]
<div data-before="前缀">我是测试文字</div>
CSS
div:before { content: attr(data-before); }
