8.31 生日快乐
8.31 生日快乐 by 雨陌 on pixiv
Github Pages 是开发者用来展示自己的项目的网站，且是静态网页，存放于名字是 username/username.github.io 的仓库，其特点是，可以通过 http://username.github.io 进行访问。
Github Pages
工作原理Hexo 生成静态网站，在 public 目录下将生成的静态网站，提交到 github 上通过链接 http://username.github.io 直接访问
系统配置需求
安装 Git 和 Node.js
通过 config 命令可以对 nodejs 默认 npm 镜像代理服务进行设置
npm config set registry https://registry.npm.taobao.org npm info underscore (如果上面配置正确这个命令会有字符串response)
Bash
git 设置用户信息
git config --global user.name [username] git config --global user.email [email] git config --list
Bash
安装 Hexo
$ d: $ mkdir /hexo $ cd /hexo $ npm install hexo-cli -g
Bash
初始化博客
搭建
$ e: $ hexo init blog $ cd blog $ npm install $ npm install hexo-deployer-git --save #安装git扩展
Bash
常用命令
$ hexo g == hexo generate #生成静态文件，在 public 文件夹下 $ hexo s == hexo server #启动本地 Web 服务，用于预览 $ hexo d == hexo deploy #部署到远程仓库：github 等
Bash
安装主题
下载主题
注：根据主题的要求安装相关插件
$ hexo clean $ git clone https://github.com/ahonn/hexo-theme-even.git themes/even
Bash
启用主题
修改 blog 下的_config.yml 配置文件中的 theme 属性，将其设置为 even。 重新生成静态文件
部署到 github page
在配置文件_config.xml 中作如下修改：
deploy: type: git repo: git@github.com:jiji262/jiji262.github.io.git branch: master
YAML
执行
$ hexo d
Bash
绑定独立域名
Github 仓库 –> Setting –> GitHub Pages –> Custom domain
进入 source 目录下，添加 CNAME 文件
$ cd source/ $ touch CNAME $ vim CNAME #你的域名
Bash
页面文章数量
除首页外的其他页面的文章数量设置可参考下列插件的设置：
hexo-generator-archive hexo-generator-tag hexo-generator-category
添加文章最后更新时间
修改（博客主目录）/themes/even/layout/_macro/post.swig 文件，在…标签后或标签里面添加如下代码：
{%if post.updated and post.updated > post.date%} <span class="post-updated"> &nbsp; | &nbsp; {{ __('post.updated') }} <time itemprop="dateUpdated" datetime="{{ moment(post.updated).format() }}" content="{{ date(post.updated, config.date_format) }}"> {{ date(post.updated, config.date_format) }} </time> </span> {% endif %}
HTML
按自己主题条件判断是否需要根据博客配置文件中的 language 参数修改对应的语言配置文件（博客主目录）/themes/even/languages/default.yml。
post: updated: 更新于
YAML
修改主题配置文件（博客主目录）/themes/even/_config.yml，增加一行
display_updated: true
YAML
写文章的时候可以直接在 md 文件开头设置更新时间
updated: 2018-01-01 12:00:00
YAML
若没有这参数的话将会显示 md 文件的修改日期。
（完）
