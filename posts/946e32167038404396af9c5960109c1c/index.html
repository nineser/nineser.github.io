居中是页面布局中最为常见的一种布局形式，其实现方法多种多样。
水平居中
text-align 应用于块级元素，
不仅影响该元素中的文本行相互之间的对齐方式，还影响具有内联特性的其它元素。
.box{ text-align: center; } .child{ display: inline-block; }
CSS
元素的 7 项水平属性的总和等于父元素的 width。
因此，对块级子元素设置具体的 width 值后，让其左右外边距值为 auto，
可以令浏览器将减去子元素 width 值后的父元素的 width 剩余空间平均分配到子元素的两边，
从而达到水平居中的目的。
.child{ width: 80%; margin-left: auto; margin-right: auto; }

半行间距等于 (line-height - font-size) / 2，
因此把 line-height 设置为需要的 box 的高度也可以实现单行文字的垂直居中。
.center-text-trick { height: 100px; line-height: 100px; white-space: nowrap; }

图片（line-height）
IE7+
思路：
利用vertical-align:middle将 img 的高度的中心点与父元素的基线往上再加 1/2 父元素中字母 X 的高度后 所在的点对齐。可以想象成，图片后面有个 X ，图片的水平中心线跟 X 的水平中心线对齐。
由于字符的下沉特性，图片和 X 的垂直中心线穿过的并不是其所在 line box 的垂直中心线，而是偏下一点。
解决方法是，让字符的垂直中心线和 line box 的垂直中心线共一条线。要么给 X 设置vertical-align:middle，要么设置 font-size: 0。
由于字母 X 在这里是 img 周围的隐匿空白节点或者换行符，无法选中，又不会受继承的 vertical-align 的影响，因此我们只能设置父容器 font-size: 0，让其继承给 X。
此时，图片与 X 这两个 inline box 仅仅是在他们所在的行框中对齐了，我们还需要将这个行框在父容器里对齐，方法是将行框的高度设置成跟父容器的高度一致。
因为父容器里面只有图片和 X 所在的这一个行框，所以我们可以用 line-height 代替 height 值。
.box{ /*用line-height代替所需要的高度。*/ line-height: 240px; /*将容器的基线、中线等所有线合为*/ font-size: 0; } img { vertical-align: middle; }
