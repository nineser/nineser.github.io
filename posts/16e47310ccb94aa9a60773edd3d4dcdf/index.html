绘制步骤
设置属性
closePath() 和 stroke()
stroke()：绘制已定义的路径
closePath()：创建从当前点回到起始点的路径
特别注意：
1.
有时候，stroke 先还是 closePath 很重要
绘制扇形的时候，如果回到圆心的路径没有手动定义，则应先闭合路径。反之，皆可。
2.
closePath() 的意思不是结束路径，而是关闭路径
它会试图从当前路径的终点连一条路径到起点，让整个路径闭合起来。
3.
但是，这并不意味着它之后的路径就是新路径了
它与 beginPath() 几乎没有关系，也不能代替 beginPath()
不要企图通过闭合现有路径来开始一条新路径
而 使用 beginPaht() 开始一条新路径，以前的路径也不会闭合
moveTo和lineTo
moveTo()：把路径移动到画布中的指定点，不创建线条
lineTo()：添加一个新点，然后在画布中创建从该点到最后指定点的线条
画布状态
堆栈数据结构存储
ctx.save(); //存储当前画布状态
ctx.restore(); // 返回之前保存过的路径状态和属性

文字相关属性
font :
font 属性设置或返回画布上文本内容的当前字体属性。
font 属性使用的语法与 CSS font 属性相同。
textAlign :
start => 默认。文本在指定的位置开始。
end => 文本在指定的位置结束。
center => 文本的中心被放置在指定的位置。
left => 文本左对齐。
right => 文本右对齐。
textBaseline :
alphabetic => 默认。文本基线是普通的字母基线。
top => 文本基线是 em 方框的顶端。
hanging => 文本基线是悬挂基线。
middle => 文本基线是 em 方框的正中。
ideographic => 文本基线是表意基线。
bottom => 文本基线是 em 方框的底端。
fillText()
text => 规定在画布上输出的文本。
x => 开始绘制文本的 x 坐标位置（相对于画布）。
y => 开始绘制文本的 y 坐标位置（相对于画布）。
maxWidth => 可选。允许的最大文本宽度，以像素计。
strokeText()
text => 规定在画布上输出的文本。
x => 开始绘制文本的 x 坐标位置（相对于画布）。
y => 开始绘制文本的 y 坐标位置（相对于画布）。
maxWidth => 可选。允许的最大文本宽度，以像素计。
// 描边 context.strokeText(text,x,y,maxWidth); //填充 context.fillText(text,x,y,maxWidth);

旋转字
//存储当前画布状态 ctx.save(); ctx.font = '55px conlosas'; ctx.textBaseline = 'middle'; //将画布中心移动到字体的位置 ctx.translate(50, 50); // 旋转画布ctx.fillText('LaiJunJie ChenQiuWen', 0, 0); ctx.rotate((45 * Math.PI) / 180); // 恢复当前画布状态 ctx.restore();

绘制图片
drawImage()
定位图像位置 => drawImage(img,x,y);
定位图像位置、宽高 => drawImage(img,x,y,width,height);
drawImage(img,sx,sy,swidth,sheight,x,y,width,height);
img => 规定要使用的图像、画布或视频。
sx => 可选。开始剪切的 x 坐标位置。
sy => 可选。开始剪切的 y 坐标位置。
swidth => 可选。被剪切图像的宽度。
sheight => 可选。被剪切图像的高度。
x => 在画布上放置图像的 x 坐标位置。
y => 在画布上放置图像的 y 坐标位置。
width => 可选。要使用的图像的宽度。（伸展或缩小图像）
height => 可选。要使用的图像的高度。（伸展或缩小图像）
var img = new Image(); img.src = 'images/1.png'; img.onload = function() { ctx.drawImage(img, 20, 20); };
