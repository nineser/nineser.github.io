关于导航
使用 float 布局，尽量不使用 inline-block，因为 inline-block 元素在 IE8 下，有时会出现 padding 重叠的问题。
搜索框圆角的兼容（IE8）
引入 对应 htc 文件，http://css3pie.com/
引入的路径相对于 HTML 文件，非样式文件
根据情况添加 z-index、position 属性
background-size 的兼容（IE8）
这里对 background-size 的兼容不支持数值， 即不支持诸如：background-size:50px。 只支持如 cover、contain…
.ele{ background-size: cover; /*路径相对于HTML文件*/ behavior: url(plugin/background-size.htc)\9; }

box-shadow 的兼容（IE8）
使用 IE 滤镜对阴影进行模拟，但有一个缺点是，无法实现有透明度的颜色。
.shadow { -moz-box-shadow: 3px 3px 4px #000; -webkit-box-shadow: 3px 3px 4px #000; box-shadow: 3px 3px 4px #000; /* For IE 8 */ -ms-filter: "progid:DXImageTransform.Microsoft.Shadow(Strength=4, Direction=135, Color='#000000')"; /* For IE 5.5 - 7 */ filter: progid:DXImageTransform.Microsoft.Shadow(Strength=4, Direction=135, Color='#000000'); }

关于 max-width
如果针对 td 中的 img 元素设置 max‐width: 100%，在 IE 和 Firefox 会不奏效，而在 Chrome 中却是可以的。解决方式是需要给 table 设置 table‐layout: fixed。
