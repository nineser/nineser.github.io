border-width
IE7 除外下，有以下属性值
thin: 1px; medium(default): 3px; thick: 5px;
CSS
3px 为默认值的原因是， 一般认为是为了满足 border-style:double 的最低使用要求
border-width 不支持百分比值，因为在生活中 border 一般不会随着物体面积的大小的改变而同比例改变，例如：大型电视显示屏与手机屏幕的边框都差不多大小。
border-style
solid、dashed、dotted
实现圆(IE7、IE8)
.dotted { width: 150px; height: 150px; border: 150px dotted #cd0000 } .box { width: 150px; height: 150px; overflow: hidden; }
CSS
double
计算规则: 大于 3px 时生效 2 条边框相同大小，剩余的为边框之间的距离。
实现三横杠
{ width: 120px; height: 20px; border-top: 60px double; border-bottom: 20px solid; }
CSS
border-color 与 color
边框的颜色值，默认为 color 的颜色值。
实现添加按钮
<a href="" class="add">+</a>
HTML
.add { color: #ccc; transition: color .35s; border: 1px solid;text-decoration: none; } .add:hover { color: #06c; }
CSS
solid 构建三角和梯形
四色板：
.box{ width: 100px; height: 100px; border: 100px solid; border-color: red green blue orange; }
CSS
通过改变 .box 的长宽，以及边框颜色 为 transparent;可以得到三角形或梯形。
border 与等高布局
原理： 用 border 模拟一列，将内容浮动至 border 内
<div class="box floatfix"> <nav class="column1"> <p>这是第一栏</p> <p>这是第一栏</p> <p>这是第一栏</p> </nav> <section class="column2"> <p>这是第二栏</p> <p>这是第二栏</p> </section> </div>
HTML
.box { width: 300px; border-left: 200px solid red; background-color: gray; } .column1 { width: 200px; float: left; margin-left: -200px; } .floatfix {*zoom: 1;} .floatfix:after {content: '';display: table;clear: both;}
CSS
border-color 与 background-position
backgroud-positon 默认相对左上角定位
在 CSS2 下实现相对于右侧定位
.box{ width: 400px; height: 400px; border-right: 50px solid transparent; background-image: url(favicon.ico); background-position: 100% 40px; background-repeat: no-repeat; }
CSS
为 png 图标选择颜色
原理： 用父元素包含 png 图标。 用 border 模拟出一个透明区域，将图片的投影定位到此区域。 通过定位，将原始图片移出父元素内容区域外，然后隐藏。
