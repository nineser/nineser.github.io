link 和 @import 的区别
来源的差别
link 是 html 标签，除了可以引用 CSS 文件，还可以引用其它文件
@import 是 CSS 提供的一种方式，只能用于加载 CSS 文件
加载顺序的差别
页面被浏览时，link 引用的 CSS 会被加载
页面全部被下载完成后，@import 引用的 CSS 才会被加载
兼容性的差别
link 标签所有版本的浏览器均支持
@import 仅支持 IE5+ 以上浏览器
dom 操作的差别
Javascript 无法控制 @import 引入的 CSS 样式
字体的引用
字体文件分为 四种，兼容不同浏览器
eot
svg
ttf
woff
@font-face { font-family: 'FamilyName'; src: url('path.eot'); src: url('path/to/_.eot?#iefix') format('embedded-opentype'), url('path/to/_.woff') format('woff'), url('path/to/_.ttf') format('truetype'), url('path/to/_.svg#FamilyName') format('svg'); } div { font-family: FamilyName; }

透明度
rgba\(\)和 opacity 都能实现透明效果。
opacity 作用于元素，以及元素内的 所有内容的透明度，
rgba\(\)只作用于元素的颜色或其背景色。
因此，设置 rgba 透明的元素的子元素不会继承透明 效果！
改变文档背景色
document.documentElement.style.backgroundColor = "#000"
