渲染模式
怪异：浏览器自行解析
标准：按 W3C 标准解析
<!DOCTYPE> 非 HTML 标签，指示浏览器页面使用哪个 HTML 版本进行编写的指令。
在 HTML 4.01 中，<!DOCTYPE> 声明引用 DTD，因为 HTML 4.01 基于 SGML。 DTD 规定了标记语言的规则，这样浏览器才能正确地呈现内容。
HTML5 不基于 SGML，所以不需要引用 DTD。
SGML：标准通用标记语言

列表项标志
list-style 只能应用于 display 的值为 list-item 的元素，可被继承。
1.
应用在 ol 或 ul，通过继承，所有的 li 都能具有设置的 list-style 样式。
2.
给某一个 li 设置 list-style 样式，可覆盖从父级继承的 list-style 样式
list-style-type
list-style-image
list-style-position
